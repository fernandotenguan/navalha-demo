# NAVALHA · Clube de Barbear

Site fictício de barbearia premium (Vila Madalena, São Paulo), construído como peça de portfólio.

**Demo ao vivo:** https://fernandotenguan.github.io/navalha-demo/

## Features

- Landing page de página única com navegação por âncoras (Serviços, A Casa, Depoimentos, Contato)
- Herói tipográfico massivo com foto integrada às letras
- Lista editorial de serviços com preços e imagem flutuante que segue o cursor (desktop)
- Animações de entrada em cascata no scroll (IntersectionObserver, GPU-friendly)
- Faixa marquee contínua, textura de grão de filme e fotos em tratamento duotone
- Mapa integrado (Google Maps embed) e agendamento via WhatsApp
- Totalmente responsivo (mobile-first) e com suporte a `prefers-reduced-motion`
- Zero dependências e zero build: HTML + CSS + JS puros, hospedagem gratuita

## Stack

HTML5, CSS3, JavaScript vanilla. Fontes: Fraunces + Archivo (Google Fonts). Fotos: Unsplash (licença Unsplash).

## Rodar localmente

Abra `index.html` no navegador. Não há build nem dependências.
